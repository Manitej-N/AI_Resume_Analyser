# 📄 AI Resume Analyser

Give it your resume and a job description — get back a match score, gaps you're missing, and honest suggestions to improve.

Built as a hands-on project to bridge Python fundamentals with real-world AI application development: PDF parsing, LLM prompting, and structured output — no fluff, just working code.

## Tech Stack
- Python
- Streamlit (UI)
- OpenAI/Gemini API (analysis engine)
- pdfplumber (PDF text extraction)

## Status
🚧 In progress — Phase 1: PDF extraction

## Why this exists
Resumes get filtered by algorithms before a human ever sees them. This tool tries to show you what that algorithm might be thinking.
